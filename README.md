# 컴활 1급 필기 CBT 모의고사

컴퓨터활용능력 **1급 필기**를 실제 CBT처럼 연습하는 모의고사 웹앱입니다.

## 특징
- **실전 60문항**(과목당 20) · 60분 타이머 · 자동 제출
- 문제 **팔레트**(번호판)로 이동, **다시 볼 문제** 플래그
- **과락·합격 자동 판정** — 한 과목이라도 40점 미만이면 과락, 과락 없이 평균 60점 이상이면 합격
- **오답노트** — 틀린 문제·정답·해설 정리
- 하프(30문항)·과목별(20문항) 모드 제공

## 문제 은행
문제는 [컴활 1급 필기 개념게임](https://hongyul67-cpu.github.io/comhwal/)과 **공유**합니다.
(`hongyul67-cpu.github.io/comhwal/data/*.js`를 불러와 출제 — 개념게임에 문제를 추가하면 CBT도 함께 풍부해집니다.)

## 결과 제출 (선생님용)
`result-collector` 방식. 학생에게 아래 형태의 링크로 배포하면 결과가 구글시트로 모입니다.
```
https://hongyul67-cpu.github.io/comhwal-cbt/?rc=<AppsScript exec URL>&cls=1,2,3&max=40
```
제출 버튼은 위 링크로 접속했을 때만 결과 화면에 나타납니다.

## 실행
`index.html`을 열면 됩니다. (GitHub Pages 배포 시 저장소 루트에 그대로)
